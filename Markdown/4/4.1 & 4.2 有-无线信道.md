# 4.1 无线信道

无线信道是指利用**电磁波**在自由空间（如大气、真空、海水）中传输信号的通道


### 4.1.1 两个核心概念：频率与带宽

在讨论无线信道前，必须分清这两个词

- **信道频带**：指这个“通道”本身允许通过的频率范围。
    
- **信号频带**：指你发送的信号实际包含的频率成分。

    ![[Pasted image 20260103211228.png]]
- 换算公式：无线电波在空间以光速 $c$ 传播，波长 $\lambda$ 与频率 $f$ 的关系为：
    
    $$\lambda = c / f \quad (c \approx 3 \times 10^8 \text{ m/s})$$

    
    > **重点**：频率越高，波长越短，天线尺寸就可以做得越小
    

### 4.1.2 频段划分与用途（常考点）

电磁波频谱极宽，不同频段特性不同。课件中列出了几个重点

|**简称**|**名称**|**典型用途**|
|---|---|---|
|**MF**|中频|商用 **AM 广播**。|
|**HF**|高频|短波通信、业余无线电。|
|**VHF**|甚高频|**FM 广播**、商业电视、航空通信。|
|**UHF**|特高频|**移动电话 (手机)**、WiFi、雷达、导航。|
|**SHF/EHF**|超/极高频|卫星通信、微波中继、**毫米波**。|

### 4.1.3 无线电波的三种传播模式（核心）

电波从发射塔到你手机，走的路径主要有三类

#### 1. 地波传播 (Ground Wave)

![[Pasted image 20260103211349.png]]

- **特点**：电波沿着地球表面弯曲传播
    
- **适用**：低频电波（2MHz 以下）
    
- **典型应用**：AM 广播。因为低频波长长，能绕过地表的障碍物
    

#### 2. 天波传播 (Sky Wave)

![[Pasted image 20260103211414.png]]
- **特点**：信号往天上传，经过大气层中的**电离层**反射折回地面
    
- **适用**：短波（HF，2~30MHz）
    
- **副作用（重点）**：**多径效应**。信号可能经过多次反射到达，路径长度不同（时延不同），会导致信号忽强忽弱
    

#### 3. 视线传播 (Line-of-Sight, LOS)

![[Pasted image 20260103211505.png]]
- **特点**：电波像光一样走直线
    
- **适用**：VHF 及更高频率（30MHz 以上）
    
- **限制**：受地球表面弯曲的限制
    
- 计算公式：
    
    如果发射天线高度为 $h$，地球半径为 $r$，则最大视距 $d \approx \sqrt{8hr}$
    
    - _例：天线高 50 米，手机离它超过 50 公里可能就没信号了_
        

---

### 4.1.4 环境对传播的影响

无线电波在穿过大气层（对流层、平流层、电离层）时，会产生以下现象

- **吸收**：空气中的水分子、氧分子会吸收能量。
    
- **反射与折射**：遇到不同密度的空气层或地面障碍物会改变方向。
    
- **散射**：遇到不均匀介质（如云、雨）时，信号向四面八方散开




# 4.2 有线信道


### 1. 明线 (Open Wire)

- **形态**：平行架设在电线杆上的裸露导线 。
    
- **特点**：
    
    - **优势**：成本极低，架设简单。
        
    - **劣势**：信号损耗大，极易受天气（雷雨、大风）和电磁干扰影响。
        
    - **现状**：目前在现代通信网络中已基本被淘汰，仅见于偏远地区的电话线或电力线。
        

### 2. 双绞线 (Twisted Pair)

![[Pasted image 20260103211626.png]]
- **结构**：由两根互相绝缘的导线按一定规则绞合而成 。
    
- **分类**：
    
    - **非屏蔽双绞线 (UTP)**：我们常用的网线（如五类、六类线）。
        
    - **屏蔽双绞线 (STP)**：外层多了一层金属屏蔽层，抗干扰能力更强 。
        
- **特性**：
    
    - **带宽**：几十 kHz 到 100 MHz 。
        
    - **用途**：价格低廉，广泛用于电话交换网的**本地环路**、**局域网**及综合布线 。
        
    - **为什么绞合？** 绞合可以抵消相邻导线产生的电磁干扰。

### 3. 同轴电缆 (Coaxial Cable)

![[Pasted image 20260103211648.png]]
- **结构**：由中心导线、绝缘层、外层网状导体和保护套组成，内外导线共用一根轴线 。
    
- **特性**：
    
    - **抗干扰**：由于电磁场被封闭在内外导体之间，**辐射损耗小**，且几乎不受外界电磁干扰 。
        
    - **带宽**：比双绞线更宽，支持更高的传输速率 。
        
- **用途**：最典型的应用是**有线电视网络 (CATV)** 。

### 4. 光纤 (Optical Fiber) —— **通信的“高速公路”**
![[Pasted image 20260103211658.png]]
- **结构**：利用玻璃或塑胶纤维传输光信号，光波在内芯和包层界面产生**全反射**，从而束缚在芯线中传输 。
    
- **四大核心优点 (必考点)** ：
    
    1. **频带宽，容量巨大**：理论潜力可达 2×1013 Hz 。
        
    2. **损耗极低**：可以实现几百公里的无中继传输 。
        
    3. **抗干扰性强**：光信号不导电，完全不受电磁干扰，且保密性好 。
        
    4. **资源丰富**：原材料是二氧化硅（沙子），价格廉价且质量轻 。
        
- **用途**：现代互联网主干网、长途电话网的核心传输媒质 。

### 💡 总结对比表

|媒质类型|信号形式|主要优点|主要缺点|典型应用|
|---|---|---|---|---|
|**双绞线**|电信号|便宜、柔软、易布线|容易受干扰、传输距离短|电话、局域网|
|**同轴电缆**|电信号|抗干扰好、带宽较宽|成本较高、线缆较硬|有线电视|
|**光纤**|光信号|**带宽极宽、损耗极低、抗磁扰**|熔接技术要求高、质脆|互联网骨干网|
