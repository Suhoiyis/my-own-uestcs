# 1. 什么是码间串扰 (ISI)？

直观理解：

想象你在一个回声很大的空房间里说话。如果你说得太快，上一个字的**回音（拖尾）**还没消失，下一个字就已经出来了，声音就会混在一起，听不清。

在通信中，信号脉冲在传输时也会产生“拖尾”。当脉冲一个接一个紧挨着发时，前面脉冲的拖尾会“甚至”到后面脉冲的位置上，这就叫码间串扰。

物理原因

1. **理想很丰满**：我们在 6.1 节学的基带信号（矩形脉冲），边缘陡峭，这就意味着它的**频带宽度是无限的**。
    
2. **现实很骨感**：实际的信道（电缆、光纤等）带宽都是**有限的**（相当于一个低通滤波器）。
    
3. **结果**：当无限带宽的信号通过有限带宽的信道时，高频分量被切掉，时域上的矩形脉冲就会变“胖”，产生长长的**拖尾**，干扰相邻的码元。

# 2. 数学定性分析（必考公式）

课件第 49-50 页用一个经典的公式把接收到的信号拆解得清清楚楚。

假设我们发送一串脉冲 $\sum a_n \delta(t - nT_B)$，经过系统（发送滤波器+信道+接收滤波器）的总冲激响应为 **$h(t)$**。

那么接收端的抽样值 $r(kT_B + t_0)$ 可以拆分为三部分：

$$r_k = \underbrace{a_k h(t_0)}_{\text{1. 本码元信息}} + \underbrace{\sum_{n \neq k} a_n h[(k-n)T_B + t_0]}_{\text{2. 码间串扰 (ISI)}} + \underbrace{n_R(kT_B + t_0)}_{\text{3. 噪声}}$$

- **第 1 项（有用信号）**：这是我们想要得到的第 $k$ 个码元的值。理想情况下，我们希望 $h(t_0)=1$。
    
- **第 2 项（ISI 大反派）**：这是**除第 $k$ 个码元以外**的所有其他码元（$n \neq k$）在当前时刻产生的**余波/拖尾**的总和。
    
- **第 3 项（噪声）**：由于热噪声等引起的随机干扰。

### 3. ISI 的危害

课件第 51 页讲到了判决规则：

- 如果接收值 $r_k > V_d$（门限），判为“1”；
    
- 如果接收值 $r_k < V_d$，判为“0”。
    

ISI 的坏处：

ISI 会在这个判决过程中“捣乱”。

- 如果你发的是“1”（高电平），但前后码元的拖尾（ISI）正好是负的，它们会把你的电平**拉低**。
    
- 如果 ISI 足够大，甚至可能让原本的“1”跌落到门限 $V_d$ 以下，导致接收端**误判为“0”** 3。
    

> **结论**：ISI 是导致误码的主要原因之一。哪怕没有外界噪声，光是 ISI 自己就能导致通信失败。

# 4. 怎么消除 ISI？（引出下一节）

既然 ISI 是因为 $h(t)$ 有“拖尾”造成的，那我们能不能设计一种神奇的 $h(t)$ 波形，让它的拖尾**恰好**在我们要抽样的时刻等于 0 呢？

- **理想状态**：如果 $h(t)$ 在 $t=0$ 时有值，而在 $t=T_B, 2T_B, ...$ 这些采样时刻**全部为 0**，那 ISI 项不就消失了吗？
    

这就是著名的 **奈奎斯特第一准则 (Nyquist's First Criterion)**，也就是下一节 **6.4 无码间串扰的基带传输特性** 要讲的核心内容。

---

### 📝 6.3 总结笔记

1. **成因**：信道带宽有限，导致矩形脉冲时域展宽（变胖），产生拖尾。
    
2. **公式结构**：接收值 = 信号 + **ISI** + 噪声。
    
3. **目标**：为了无误码传输，我们必须想办法让 ISI 项为 0。
