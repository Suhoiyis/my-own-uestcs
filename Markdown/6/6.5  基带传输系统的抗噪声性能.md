
# 6.5.1 二进制双极性基带系统的抗噪性能

这一节的核心任务是：**计算在噪声干扰下，我们判错码的概率是多少？（即误码率 $P_e$）**

## 1. 场景设定：信号与噪声的博弈

课件第 87 页展示了分析模型：
![[Pasted image 20260104195129.png]]
- **信号**：我们发送的是**双极性**信号。
    
    - 发送 "1" $\rightarrow$ 接收电平为 **$+A$**。
        
    - 发送 "0" $\rightarrow$ 接收电平为 **$-A$**。
        
- **噪声**：信道中叠加了高斯白噪声 $n(t)$，经过接收滤波器后变成了 **$n'(t)$**。
    
    - $n'(t)$ 是一个均值为 0，方差为 $\sigma_n^2$ 的高斯随机变量
        

最终抽样时刻的总电压 $x$：

$$x = \text{信号} + \text{噪声}$$

- 发 "1" 时：$x = +A + n'$
    
- 发 "0" 时：$x = -A + n'$
    

---

### 2. 直观理解：为什么会出错？

请看下面的“双峰图”（课件第 90 页）

![[Pasted image 20260104195216.png]]

- 右边的山峰（以 $+A$ 为中心）代表发送 "1" 时电压的概率分布。
    
- 左边的山峰（以 $-A$ 为中心）代表发送 "0" 时电压的概率分布。
    

**判决规则**：我们在中间设一道门坎 **$V_d$（判决门限）**。

- 电压 $> V_d$，判为 "1"。
    
- 电压 $< V_d$，判为 "0"。
    

**出错的情况**：

- **发 "1" 错判为 "0"**：虽然发的是 $+A$，但噪声 $n'$ 恰好是一个很大的负值，把总电压拉得太低，跌过了门坎 $V_d$（对应右边山峰在 $V_d$ 左侧的阴影面积）
    
- **发 "0" 错判为 "1"**：虽然发的是 $-A$，但噪声 $n'$ 恰好是一个很大的正值，把总电压推得太高，越过了门坎 $V_d$
    

---

### 3. 最佳门限：门坎设在哪里最好？

直觉告诉我们，既然 $+A$ 和 $-A$ 是对称的，如果 "0" 和 "1" 出现的概率相等（$P(1)=P(0)=0.5$），那门坎肯定设在正中间最公平。

数学推导证明了这一点（课件第 94 页）：

- 当 **$P(1) = P(0) = 0.5$** 时，**最佳门限 $V_d = 0$**
    
- 这就解释了为什么双极性系统不需要调节判决电平，直接用 0V（地）做参考就行。
    

---

### 4. 误码率公式推导（必考结论）

当门限 $V_d = 0$ 时，误码率 $P_e$ 计算如下：

1. 公式推导：
    
    $$P_e = P(1) \times P(\text{1错判为0}) + P(0) \times P(\text{0错判为1})$$
    
    由于对称性，两种错判概率相等，最终简化为求其中阴影部分的面积。
    
2. 最终公式（使用互补误差函数 $\text{erfc}$）5：
    
    $$P_e = \frac{1}{2} \text{erfc}\left( \frac{A}{\sqrt{2}\sigma_n} \right)$$
    
    - **$A$**：信号幅度（半幅值）。
        
    - **$\sigma_n$**：噪声的标准差（噪声功率开根号）。
        
    - **$\text{erfc}(x)$**：这是一个衰减极快的函数。$x$ 越大（信噪比越高），函数值越趋近于 0（误码率越低）。

### 💡 6.5.1 核心笔记总结

- **模型**：$+A / -A$ 加上高斯噪声。
    
- **门限**：对于等概信号，最佳判决门限是 **$0$**。
    
- **抗噪性能**：取决于 **信噪比 ($A/\sigma_n$)**。信号幅度 $A$ 越大，或者噪声 $\sigma_n$ 越小，误码率 $P_e$ 就越低。

# 6.5.2 单极性基带系统的抗噪性能

![[Pasted image 20260104201423.png]]

## 1. 场景设定：不对称的博弈

- **信号**：
    
    - 发送 "1" $\rightarrow$ 电平 **$+A$**。
        
    - 发送 "0" $\rightarrow$ 电平 **$0$**。
        
- **噪声**：同样的零均值高斯白噪声 $n'(t)$。
    
- **接收波形**：
    
    - 发 "1"：$x = A + n'$
        
    - 发 "0"：$x = 0 + n'$
        

## 2. 最佳门限：这就不能是 0 了

对于双极性（$+A/-A$），中间点是 0。

对于单极性（$+A/0$），直觉告诉我们，中间点应该是 $A/2$。

结论：

当发 "1" 和发 "0" 概率相等时，最佳判决门限 $V_d = A/2$

> **注意**：这就带来了一个缺点——判决门限与信号幅度 $A$ 有关。如果信号在传输中衰减了（$A$ 变小），接收端的门限 $A/2$ 也得跟着调，否则误码率会飙升。而双极性系统的门限永远是 0，不受幅度影响 

### 3. 误码率公式

有了门限 $V_d = A/2$，信号距离门限的“安全距离”就是 $A - A/2 = A/2$。

直接套用 erfc 公式：

$$P_{e(\text{单})} = \frac{1}{2} \text{erfc}\left( \frac{A/2}{\sqrt{2}\sigma_n} \right) = \frac{1}{2} \text{erfc}\left( \frac{A}{2\sqrt{2}\sigma_n} \right)$$

![[Pasted image 20260104201504.png]]

---

## 🆚 终极 PK：双极性 vs 单极性

现在我们把两个公式放在一起看，胜负一目了然：

| **系统类型** | **信号电平** | **噪声容限 (信号到门限的距离)** | **误码率公式 Pe​ (括号里越大越好)**                                             |
| -------- | -------- | ------------------- | ------------------------------------------------------------------- |
| **双极性**  | $+A, -A$ | **$A$** (大)         | $\frac{1}{2} \text{erfc}\left( \frac{A}{\sqrt{2}\sigma_n} \right)$  |
| **单极性**  | $+A, 0$  | **$A/2$** (小)       | $\frac{1}{2} \text{erfc}\left( \frac{A}{2\sqrt{2}\sigma_n} \right)$ |

1. 抗噪性能对比

- **双极性胜出**：你可以看到，双极性公式括号里的分子是 $A$，而单极性是 $A/2$。
    
- **物理意义**：双极性系统的“抗干扰空间”（信号到门限的距离）是单极性的 **2 倍**。所以，在同样的噪声 $\sigma_n$ 下，双极性的误码率要低得多。

![[Pasted image 20260104201540.png]]
    

### 2. 功率代价对比

如果我们要求两者的误码率相同，单极性系统需要付出多大的代价？

- 为了让单极性的 $A/2$ 赶上双极性的 $A$，单极性的幅度 $A$ 必须加倍。
    
- 幅度加倍意味着**功率（平方）要增加 4 倍**，即增加 **6dB**。
    
- 但考虑到单极性有一半时间发 0（不耗电），平均功率实际增加 **3dB**。
    

> **结论**：**双极性系统比单极性系统具有 3dB 的信噪比优势。** 也就是它更省电、更抗噪。

---

# 📝 6.5 总结笔记

- **计算核心**：$P_e$ 取决于 **(信号到门限的距离) / (噪声标准差)**。距离越远，误码越低。
    
- **单极性劣势**：门限是 $A/2$，距离只有 $A/2$，而且门限随信号强度漂移，不稳定。
    
- **双极性优势**：门限是 0，距离是 $A$，门限固定且抗噪强。
