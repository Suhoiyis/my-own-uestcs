
# 7.1.1 二进制振幅键控 (2ASK / OOK)

## 1. 核心原理

**2ASK (Binary Amplitude Shift Keying)**：用二进制信号（0 或 1）来控制载波的幅度。

- 发送 "1" $\rightarrow$ 发送载波（有幅度）。
    
- 发送 "0" $\rightarrow$ 不发送载波（幅度为 0）或发送低幅度载波。
    

**OOK (On-Off Keying)**：这是 2ASK 的一种特例，也是最常用的形式。

- **规则**：就像开灯关灯一样。
    
    - 信息 "1"：有载波（On）。
        
    - 信息 "0"：无载波（Off）
        
![[Pasted image 20260104202316.png]]

## 2. 数学表达式

OOK 信号的时域表达式非常简单：

$$e_{\text{2ASK}}(t) = s(t) \cdot \cos(2\pi f_c t)$$

- **$s(t)$**：单极性非归零矩形脉冲序列（1 或 0）。
    
- **$\cos(2\pi f_c t)$**：高频载波。
    

## 3. 产生方法（怎么做？）

有两种常见的产生 OOK 信号的方法：

1. 模拟调制法（乘法器）：
    
    直接用基带信号 $s(t)$ 乘以载波 $\cos(\omega_c t)$。
    
    - $1 \times \cos = \cos$
        
    - $0 \times \cos = 0$
        
2. 键控法（开关电路）：
    
    用 $s(t)$ 控制一个开关。$s(t)=1$ 时开关闭合，输出载波；$s(t)=0$ 时开关断开，输出 0。

![[Pasted image 20260104205558.png]]
    

## 4. 解调方法（怎么收？）

接收端要把高频载波去掉，还原出 0 和 1。有两种解调方式 3：

#### **(1) 非相干解调（包络检波法）**

这是最简单的，就像收音机一样。

- **流程**：带通滤波器 $\rightarrow$ **全波整流器 + 低通滤波器**（提取包络）$\rightarrow$ 抽样判决。
    
- **优点**：简单，不需要提取同频同相的载波。
    
- **缺点**：抗噪声性能稍差（有门限效应）。
    

#### **(2) 相干解调（同步检测法）**

- **流程**：带通滤波器 $\rightarrow$ **乘法器**（乘以本地载波）$\rightarrow$ 低通滤波器 $\rightarrow$ 抽样判决。
    
- **要求**：接收端必须产生一个与发送端**同频同相**的本地载波（同步信号）。
    
- **优点**：抗噪性能好。
    
- **缺点**：电路复杂，载波同步难做。
    
![[Pasted image 20260104205639.png]]

## 5. 功率谱与带宽（考点

OOK 信号本质上是基带信号频谱的**线性搬移**。

- **功率谱**：基带信号 $s(t)$ 的功率谱被搬移到了 $+f_c$ 和 $-f_c$ 处。
    
- 带宽 $B_{\text{OOK}}$：
    
    $$B_{\text{OOK}} = 2f_B = 2R_B$$
    
    - 也就是：OOK 信号的传输带宽是基带信号带宽（$R_B$）的 **2 倍**（因为有上下两个边带）。

## 📝 7.1.1 总结笔记

- **OOK**：有载波传 1，无载波传 0。
    
- **带宽**：$2R_B$（双边带）。
    
- **解调**：包络检波（简单）、相干解调（性能好）。
    
- **地位**：最基础的调制，虽然简单但抗干扰能力差（幅度容易受噪声影响），常用于光纤通信。


# 7.1.2 二进制频移键控（2FSK）

## 1. 核心原理与表达式

**定义**：利用载波**频率的变化**来传输数字信息。

- 发送 "1" $\rightarrow$ 发送频率为 $f_1$ 的载波。
    
- 发送 "0" $\rightarrow$ 发送频率为 $f_2$ 的载波
    

数学表达式：

$$e_{2FSK}(t) = \begin{cases} A \cos(\omega_1 t + \phi_n), & \text{发 "1"} \\ A \cos(\omega_2 t + \theta_n), & \text{发 "0"} \end{cases}$$

视角的转换（重要理解）：

2FSK 信号其实可以看作是两个互补的 OOK 信号的叠加 2。

- 一个频率为 $f_1$ 的 OOK 信号（传 "1" 时有，传 "0" 时无）。
    
- 一个频率为 $f_2$ 的 OOK 信号（传 "0" 时有，传 "1" 时无）。
    
- 这种“分解视角”对理解它的频谱和解调非常有帮助。

![[Pasted image 20260104210018.png]]

---

## 2. 产生方法：相位连不连续？

2FSK 的产生主要有两种方法，它们的区别在于**相位是否连续**，这直接影响带宽特性。

### **(1) 键控法 (Keying) —— 相位可能不连续**

- **原理**：用开关在两个独立的振荡器（$f_1$ 和 $f_2$）之间切换
    
- **缺点**：因为两个振荡器各自为政，切换瞬间相位可能会突变（不连续）。
    
- **后果**：相位跳变会产生高频分量，导致**频带展宽**（占用更多带宽）

![[Pasted image 20260104210042.png]]

### **(2) 模拟调制法 (VCO) —— 相位连续**

- **原理**：使用**压控振荡器 (VCO)**，直接用基带电压控制输出频率 5。
    
- **优点**：相位是连续变化的（CP-FSK），频谱更纯净，带宽更窄。
    
	![[Pasted image 20260104210111.png]]
---

## 3. 解调方法（三种流派）

### **(1) 非相干解调（包络检波法）**

- **原理**：利用“分解视角”，把它看作两个 OOK。
    
    - 上支路：$f_1$ 带通滤波器 + 包络检波 $\rightarrow$ 检出 "1"。
        
    - 下支路：$f_2$ 带通滤波器 + 包络检波 $\rightarrow$ 检出 "0"。
        
    - **判决**：比较两路输出电压的大小，谁大判谁
        
- **条件**：为了能把两路信号分开，两个频率必须离得够远，通常要求 $|f_1 - f_2| [cite_start]\ge 2R_B$
    

### **(2) 相干解调（同步检测法）**

- **原理**：和 OOK 类似，也是分成两路，分别乘以本地载波 $\cos \omega_1 t$ 和 $\cos \omega_2 t$ 8。
    
- **优点**：抗噪性能优于非相干。
    

### **(3) 过零检测法（2FSK 特有）**

- **原理**：频率越高，波形穿过零点的次数就越多。
    
- **流程**：限幅 $\rightarrow$ 微分 $\rightarrow$ 整流 $\rightarrow$ 计数。通过数单位时间内波形**过零点的次数**来判断是 $f_1$ 还是 $f_2$ 9。
    
- **优点**：适合数字化处理。

![[Pasted image 20260104210148.png]]

---

## 4. 功率谱与带宽（核心计算）

由于 2FSK 可以看作两个 OOK 的叠加，其频谱就是两个 OOK 频谱在频率轴上的搬移

- **频谱形状**：典型的“双峰”结构（两个骆驼峰）。
    
- 带宽公式 ($B_{2FSK}$)：
    
    $$B_{2FSK} = |f_2 - f_1| + 2R_B$$
    
    - 物理意义：两个载频的间距 + 每个 OOK 信号的带宽 11。
        
    - _注：这里使用的是卡森公式的近似估算。_
        

---

## 📝 7.1.2 总结笔记

- **2FSK**：用频率高低传 0/1。
    
- **产生**：VCO 法产生的相位连续，频谱更好。
    
- **带宽**：$|f_2 - f_1| + 2R_B$（双峰结构）。
    
- **解调**：过零检测法是它特有的黑科技。
    
- **特点**：抗干扰能力强于 2ASK（因为幅度变化不携带信息），常用于低速数据传输（如早期的电话线 Modem）。


# 7.1.3 二进制相移键控（2PSK）

## 1. 二进制相移键控 (2PSK / BPSK)

### **(1) 核心原理**

**2PSK** 利用载波的**相位**变化来传递数字信息。通常使用相位 0 和 $\pi$（180度）来对应二进制的“0”和“1”

- 发送 "0" $\rightarrow$ 相位为 0 $\rightarrow$ $\cos(\omega_c t)$
    
- 发送 "1" $\rightarrow$ 相位为 $\pi$ $\rightarrow$ $\cos(\omega_c t + \pi) = -\cos(\omega_c t)$
    

### **(2) 数学本质**

2PSK 信号可以看作是一个双极性基带信号 $s(t)$（取值为 $+1, -1$）与载波的乘积

$$e_{2PSK}(t) = s(t) \cdot \cos(\omega_c t)$$

- 当 $s(t) = +1$ 时，输出 $+\cos$
    
- 当 $s(t) = -1$ 时，输出 $-\cos$
    

### **(3) 致命缺陷：“倒 $\pi$ 现象” (Phase Ambiguity)**

2PSK 的解调必须使用相干解调（需要本地载波）。

但在实际接收端，提取本地载波时存在 180° ($\pi$) 的相位模糊。

- **后果**：接收端恢复出来的载波可能和发送端同相，也可能反相。
    
- 如果恢复的载波反相了，那么**解调出的所有 0 和 1 就会全部颠倒**（0 变 1，1 变 0）。这就是著名的**“倒 $\pi$ 现象”** 3。
    

> **结论**：因为怕反相，2PSK 在实际中几乎**没法单独使用**。

---

## 2. 救星登场：二进制差分相移键控 (2DPSK)

为了解决 2PSK 怕反相的问题，我们发明了 2DPSK。

**核心思想**：不看相位的“绝对值”，只看前后码元相位的**“相对变化”**

### **(1) 编码规则（相对移相）**

- **信息 "0"**：相位**不变**（和前一码元同相）。
    
- **信息 "1"**：相位**翻转**（和前一码元反相，跳变 $\pi$）。
    
- **好处**：即使接收端的载波反相了，"相位变没变"这个事实是不会变的。
    

### **(2) 实现方法：差分编码 + 2PSK**

2DPSK 信号的产生分两步

1. **差分编码**：把输入的绝对码 $\{b_n\}$ 变成相对码 $\{d_n\}$。
    
    - 逻辑公式：$d_n = b_n \oplus d_{n-1}$ （模二加）。
        
2. **2PSK 调制**：用这个新的相对码 $\{d_n\}$ 去进行普通的 2PSK 调制。
    

---

### 3. 2DPSK 的解调（两种大招）

#### **方法一：相干解调 + 差分译码**

先假装是 2PSK 解调（可能会反相），然后再加一级**差分译码器**（把反相纠正回来）。

- 缺点：还是需要提取载波，电路较复杂
    

#### **方法二：差分相干解调（相位比较法）—— 必考黑科技**

这种方法**不需要**提取本地载波

- **原理**：利用**前一个码元的信号**作为后一个码元的“参考载波”。
    
- 结构：
    
    $$r(t) \times r(t - T_B) \rightarrow \text{低通滤波器} \rightarrow \text{抽样判决}$$
    
    将接收到的信号延迟一个码元时间 $T_B$，然后自己乘自己。
    
    - 如果前后同相（发 "0"）：乘积为正。
        
    - 如果前后反相（发 "1"）：乘积为负。
        
- **优点**：电路极其简单，不需要复杂的载波同步电路。
    
- **缺点**：抗噪性能比相干解调略差（约差 1-3dB）。
    

---

### 4. 2PSK/2DPSK 的带宽

它们的波形本质上都是二元信号调制的正弦波，所以带宽和 2ASK 一样

$$B_{\text{2PSK}} = B_{\text{2DPSK}} = 2R_B$$

### 📝 7.1.3 & 7.1.4 总结笔记

|**特性**|**2PSK (绝对相移)**|**2DPSK (相对相移)**|
|---|---|---|
|**信息承载**|绝对相位 (0, $\pi$)|相位差 ($\Delta\phi$)|
|**致命弱点**|**倒 $\pi$ 现象** (反相)|无|
|**产生方法**|直接调制|差分编码 + 2PSK|
|**解调方法**|必须相干解调|1. 相干+差分译码<br><br>  <br><br>2. **相位比较法** (无需载波)|
|**应用**|很少单独使用|广泛应用 (因抗干扰强且不怕反相)|
