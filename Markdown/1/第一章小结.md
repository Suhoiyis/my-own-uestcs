### 1. 通信的目的与电信发明史

- **通信的目的：** 传递消息中所包含的信息
    
- **电信发明史关键节点：**
    
    - 1837年：莫尔斯发明有线电报
        
    - 1876年：贝尔发明有线电话
        
    - 1918年：调幅无线电广播、超外差接收机问世
        
    - 1936年：商业电视广播开播
        

### 2. 通信系统的模型

- **一般模型组成：** 信源 $\rightarrow$ 发送器 $\rightarrow$ 信道 $\rightarrow$ 接收器 $\rightarrow$ 信宿（以及噪声）
    
- **各部分功能：**
    
    - **发送器：** 将信源产生的信号变换成适合在信道中传输的信号
        
    - **信道：** 传输信号的物理媒质
        
    - **接收器：** 完成发送器的反变换，从带有干扰的接收信号中恢复出原始信息
        

### 3. 信号类型与形态

- **数字信号 vs 模拟信号：**
    
    - **模拟信号：** 对应模拟信源，消息是无限多种取值的连续量（如语音）
        
    - **数字信号：** 对应数字信源，消息只有有限种取值（如键盘字符）
        
- **基带信号 vs 已调信号（带通/频带信号）：**
    
    - **基带信号：** 模拟消息信号大多在零赫兹附近，未经过频谱搬移
        
    - **已调信号（带通信号）：** 借助于正弦载波，将基带信号搬移到特定频带形成的信号


### 4. 数字通信的特点

- **优点：**
    
    1. 容易再生以避免噪声积累，利于长距离传输
        
    2. 易于进行差错控制、压缩与加密处理 
        
    3. 易于多种信源数据（语音、图像等）综合传输
        
    4. 便于与计算机及网络结合
        
- **缺点：**
    
    1. 通常需要更多的带宽
        
    2. 需要复杂的同步系统
        

### 5. 通信系统分类

- **按传输媒质：** 有线通信（光纤、电缆等）和无线通信（微波、卫星等）
    
- **按信号特征：** 模拟通信系统和数字通信系统
    
- **按调制方式：** 线性调制（如AM）、非线性调制（如FM）、数字调制（如PSK）等
    
- **按复用方式：** 频分复用、时分复用、码分复用
    

### 6. 通信方式（传输模式）

- **按传输方向：**
    
    - **单工：** 只能沿一个固定方向传输（如广播）
        
    - **半双工：** 可以双向传输，但不能同时进行（如对讲机）
        
    - **全双工：** 可以同时沿两个方向传输（如电话）
        
- **按数据排列（数字通信）：**
    
    - **并行传输：** 多条信道同时传输成组的比特，速度快但成本高 
        
    - **串行传输：** 一条信道逐个比特传输，节省成本但速度较慢 
        

### 7. 信息及其度量

- **概念：** 信息是消息的内涵，是受信者原来不知而待知的内容 
    
- **单位：** 比特 (Bit) 
    
- **度量关系：** 当二进制数字 0 和 1 出现的概率相等时，传送一个二进制数字的信息量为 1 bit
    

### 8. 通信系统的主要性能指标

这是考试计算题的重点，分为两类系统来看：

| **系统类型**   | **有效性指标 (传得快/多)**               | **可靠性指标 (传得好/准)**                 |
| ---------- | ------------------------------- | --------------------------------- |
| **模拟通信系统** | **传输带宽** (有效传输频带)               | **输出信噪比** (接收端)                   |
| **数字通信系统** | **传输速率** ($R_B$ 波特率, $R_b$ 比特率) | **误码率** ($P_e$), **误比特率** ($P_b$) |
