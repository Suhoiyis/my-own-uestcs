## 1. 通信系统的一般模型

课件首先展示了一个**通用模型**，无论是有线电话还是 WiFi，都可以抽象为这 5 个部分 ：

我们可以把它看作一个流水线：

1. **信源 (Source) & 信宿 (Destination)：**
    
    - 一个是信息的发源地（如说话的人、摄像机），一个是信息的归宿（如听众、显示器）。
        
2. **发送设备 (Transmitter/发送器)：**
    
    - 它的核心任务是**匹配** 。将信源和信道匹配起来，即将信源产生的信号变换成适合在信道中传输的信号
        
    - 信源出来的原始信号（比如微弱的语音电压）通常不能直接传很远，发送设备要把这些信号进行处理（比如放大、调制），让它变得“强壮”且适合在路上传输 。
        
3. **信道 (Channel)：**
    
    - 这是信号传输的**物理媒质**，也就是信号走的“路” 。
        
    - _关键点：_ 信道中总会有**噪声 (Noise)** 混进来，这是通信系统最大的敌人 。
        
4. **接收设备 (Receiver/接收器)：**
    
    - 它的功能与发送设备相反。它的任务是从那一堆带有干扰和噪声的接收信号中，**正确恢复**出原始信息 。

## 2. 两种具体的系统模型

进一步把系统分为了 **模拟通信系统** 和 **数字通信系统**，这二者有很大的区别：

- **模拟通信系统 (Analog Communication System)：**
    
    - 传输的是模拟消息。
        
    - **系统的核心**是 **调制 (Modulation)** 和 **解调 (Demodulation)** 。
        
    - _简单理解：_ 就像把信件（信息）装进卡车（载波）里运送出去。
        
- **数字通信系统 (Digital Communication System)：**
    
    - 传输的是数字消息。
        
    - 结构比模拟系统复杂得多，包含 **格式化、信源编码、加密、信道编码** 等多个步骤 。
        
    - **优点：** 抗干扰能力强（易于再生），容易加密，且便于与计算机连接 。
