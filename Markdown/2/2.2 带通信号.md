这一节的核心目的是：**如何用简单的低频信号（基带）数学公式，来描述复杂的高频信号（带通）**

为了做到这一点，我们需要引入三个层层递进的数学工具：

1. **希尔伯特变换** (2.2.1)
    
2. **解析信号** (2.2.2)
    
3. **复包络** (2.2.3)
    

我们先从第一个工具——**希尔伯特变换**开始。

# 1. 希尔伯特变换 (Hilbert Transform)

很多同学学到这里会懵，其实它的物理意义非常简单：**“全频段 90° 相移”**。

- 定义：
    
    一个实信号 $x(t)$ 的希尔伯特变换 $\hat{x}(t)$，定义为它与 $\frac{1}{\pi t}$ 的卷积
    
    $$\hat{x}(t) = x(t) * \frac{1}{\pi t}$$
    
- 频域特性（考试必考点）：
    
    在时域做卷积太难算了，我们看频域。希尔伯特变换相当于给信号通过了一个特殊的滤波器 $H_h(f)$ 
    
    $$H_h(f) = -j \cdot \text{sgn}(f)$$
    
    - **对于正频率 ($f>0$)：** 乘以 $-j$ $\rightarrow$ 相位**滞后 90°** ($-\pi/2$)。
        
    - **对于负频率 ($f<0$)：** 乘以 $j$ $\rightarrow$ 相位**超前 90°** ($+\pi/2$)。
        
- **最经典的例子：**
    
    - $\cos(\omega_0 t)$ 的希尔伯特变换是 $\sin(\omega_0 t)$。
        
    - $\sin(\omega_0 t)$ 的希尔伯特变换是 $-\cos(\omega_0 t)$。
        
    - _你看，就是把余弦变成了正弦，推迟了 90 度。_





# 2.2.2 解析信号 (Analytic Signal)


**通过数学手段，把一个实信号变成一个“只包含正频率”的复信号，从而简化后续对带通系统的分析。**

### 1. 定义

给定一个实信号 $x(t)$，它的解析信号 $z(t)$ 定义为一个复信号，其中：

- **实部**：原实信号 $x(t)$
    
- **虚部**：原信号的希尔伯特变换 $\hat{x}(t)$
    

![[Pasted image 20260103203433.png]]

**公式：**

$$z(t) = x(t) + j\hat{x}(t)$$

### 2. 频域特性（最重要的性质）

这是解析信号存在的意义。根据课件（第 33-34 页）：

- **频谱公式**：$Z(f) = 2X(f) \cdot u(f)$
    
- **其中 $u(f)$ 是单位阶跃函数**：当 $f > 0$ 时为 1，当 $f < 0$ 时为 0。
    

**这意味着：**

1. **负频率消失**：解析信号在负频率区域（$f < 0$）的频谱强度为 **0**。
    
2. **正频率翻倍**：在正频率区域（$f > 0$），其频谱强度是原信号的 **2 倍**。
    

> **直观理解**：实信号的频谱是左右对称的（正负频率各一半能量）。解析信号把负频率的能量“搬”到了正频率上，合二为一，使得我们只需要研究正频率即可代表整个信号。

### 3. 解析信号的矢量表示（复指数形式）

解析信号可以用幅度（包络）和相位来表示：

$$z(t) = a(t)e^{j\phi(t)}$$

- 瞬时幅度（包络） $a(t)$：

	- **字面意思：** “包络”就像是一个信封（Envelope），把快速振动的信号包裹在里面。
	    
	- **通俗理解：** 想象一个跳绳的动作，绳子上下飞快地甩动（高频波形），但绳子摆动的最大高度形成的那个弧度轮廓，就是“包络”。
	    
	- **数学本质：** $a(t) = \sqrt{x^2(t) + \hat{x}^2(t)}$。它代表了信号在某一个瞬间的**幅度（强度）**。
	    
	- **为什么重要：** 在收音机（AM调制）里，声音信息就藏在这个“轮廓”里。
    $$a(t) = |z(t)| = \sqrt{x^2(t) + \hat{x}^2(t)}$$
    
    这是提取调制信号“轮廓”的关键公式。
    
- 瞬时相位 $\phi(t)$：

	- **字面意思：** 信号在某一时刻所处的“状态”或“角度”。
	    
	- **通俗理解：** 想象一个圆周运动的点。这个点现在转到了几点钟方向？是 3 点钟方向（0度）还是 12 点钟方向（90度）？这个角度就是相位。
	    
	- **数学本质：** $\phi(t) = \arctan\frac{\hat{x}(t)}{x(t)}$。
	    
	- **为什么重要：** 在 WiFi 或 5G 信号中（PSK/QAM调制），我们通过改变这个角度来代表“0”和“1”。
$$\phi(t) = \arg[z(t)] = \arctan\frac{\hat{x}(t)}{x(t)}$$
    
- 瞬时频率 $f_i(t)$：

	- **字面意思：** 信号在某一个极短瞬间的振动快慢。
	- **通俗理解：** 想象你在操场跑步。你现在的**瞬时速度**（每秒跑几米）就是你的瞬时频率。如果你跑得越来越快，频率就在增加。
	- **数学逻辑：** 它是**相位的变化率**（导数）。
	    - 公式：$f_i(t) = \frac{1}{2\pi} \frac{d\phi(t)}{dt}$
	        
	    - 如果你转圈的速度恒定，频率就是固定的；如果你忽快忽慢，频率就在变。
	        
	- **为什么重要：** 这是调频广播（FM）的核心原理。
    $$f_i(t) = \frac{1}{2\pi} \frac{d\phi(t)}{dt}$$
    

### 4. 为什么要引入解析信号？

1. **数学简化**：处理复指数 $e^{j\omega t}$ 比处理三角函数 $\cos(\omega t)$ 要简单得多。
    
2. **去除冗余**：实信号的负频率部分并不包含额外信息（它是正频率的镜像）。去除它后，分析带通系统（如无线电）时会更清晰。
    
3. **提取包络**：在解调（接收信号）时，通过解析信号可以非常方便地计算出信号的幅度变化。
    

---

### 🧠 考点提醒

在考试或做题时，如果题目给你一个 $\cos(\omega_c t)$，问它的解析信号是什么？

- **第一步**：求希尔伯特变换 $\rightarrow \sin(\omega_c t)$。
    
- **第二步**：组合 $\rightarrow \cos(\omega_c t) + j\sin(\omega_c t)$。
    
- **第三步**：利用欧拉公式简化 $\rightarrow e^{j\omega_c t}$。
    

**结论**：$e^{j\omega_c t}$ 就是 $\cos(\omega_c t)$ 的解析信号。



# 2.2.3 带通信号的三种表达式

一个带通信号（Bandpass Signal）是指频谱只在某个有限区间 $(f_1, f_2)$ 上非零的信号，通常围绕着中心频率 $f_c$ 

### 1. 包络相位表达式 (Envelope-Phase Representation)

这是最直观的表达方式，直接对应我们刚才讲的“旋转箭头” 

$$x(t) = a(t) \cos[\omega_c t + \theta(t)]$$

- **$a(t)$：** 瞬时包络，代表信号幅度的缓慢变化
    
- **$\theta(t)$：** 瞬时相位，代表相对于载波的变化
    
- **$\omega_c t$：** 载波的高频旋转部分
    

### 2. 正交表达式 (Quadrature Representation) —— **硬件实现的核心**

通过三角函数展开，我们可以把信号拆成两个相互垂直（正交）的分量

$$x(t) = x_c(t) \cos(\omega_c t) - x_s(t) \sin(\omega_c t)$$

- **$x_c(t)$：** **同相分量 (In-phase)**，与 $\cos$ 载波同向
    
- **$x_s(t)$：** **正交分量 (Quadrature)**，与 $\cos$ 正交（即 $\sin$）
    
- **关系：** $x_c(t) = a(t)\cos\theta(t)$，$x_s(t) = a(t)\sin\theta(t)$ 
    

### 3. 复包络表达式 (Complex Envelope Representation) —— **数学分析的终极工具**

这是用复数形式来表达，最为简洁

$$x(t) = \text{Re}[x_L(t) e^{j\omega_c t}]$$

- **$x_L(t)$：** 称为**复包络**或**等效基带信号** 
    
- **核心等式：** $x_L(t) = x_c(t) + jx_s(t) = a(t)e^{j\theta(t)}$ 
    

---

## 为什么要学复包络 $x_L(t)$？

这是整个第二、三章最重要的结论：

**带通信号 $x(t)$ 本质上只包含两个要素：复包络 $x_L(t)$ 和载频 $f_c$**

在频域上，$x_L(t)$ 的频谱 $X_L(f)$ 其实就是把高频的带通频谱 $X(f)$ 直接搬移到零频率（基带）位置的结果

这有什么好处？

在实际研究 WiFi、5G 等高频系统时，我们不需要在复杂的 $GHz$ 频率下算公式，只需要研究那个掉到 $0Hz$ 附近的复包络 $x_L(t)$ 就可以了





# 2.2.4 频谱搬移 (Frequency Shifting)

频谱搬移是频带通信的核心。它的本质是利用正弦波（载波）的特性，把低频信号搬移到高频，或者从高频拉回到低频。

### 1. 发送过程：调制 (Modulation) —— “穿上外衣”

为了形成一个高频的带通信号 $x(t)$，我们利用正交表达式：

$$x(t) = x_c(t) \cos(\omega_c t) - x_s(t) \sin(\omega_c t)$$

- **物理过程：**
    
    1. 准备好两个基带信号分量：同相分量 $x_c(t)$ 和正交分量 $x_s(t)$。
        
    2. 分别用它们去乘两个相互正交的载波（$\cos$ 和 $-\sin$）。
        
    3. 把结果相加。
        
- **结果：** 此时，信号的频谱已经从 0Hz 附近搬移到了 $f_c$ 附近。
    

### 2. 接收过程：解调 (Demodulation) —— “脱掉外衣”

接收端拿到的信号是高频的 $x(t)$，为了取出里面的 $x_c(t)$ 和 $x_s(t)$，我们需要进行反向操作。

根据课件第 49-50 页，最常用的方法是**相干解调**：

- **提取 $x_c(t)$：** 用 $x(t)$ 乘以 $2\cos(\omega_c t)$，然后通过一个 **低通滤波器 (LPF)**。
    
- **提取 $x_s(t)$：** 用 $x(t)$ 乘以 $-2\sin(\omega_c t)$，然后通过一个 **低通滤波器 (LPF)**。
    

---

### 🧠 为什么乘以载波再加低通就能找回原信号？（核心推导）

我们以提取 $x_c(t)$ 为例，看看这个“数学魔术”是怎么变的（参考课件第 50-51 页）：

1. **相乘：** $x(t) \times 2\cos(\omega_c t)$
    
2. 展开： 把 $x(t)$ 的正交表达式代入：
    
    $$[x_c(t) \cos(\omega_c t) - x_s(t) \sin(\omega_c t)] \times 2\cos(\omega_c t)$$
    
3. **利用三角函数公式：**
    
    - $\cos^2 A = \frac{1 + \cos 2A}{2}$
        
    - $\sin A \cos A = \frac{1}{2} \sin 2A$
        
4. 得到结果：
    
    $$x_c(t) + \underbrace{x_c(t) \cos(2\omega_c t) - x_s(t) \sin(2\omega_c t)}_{\text{这是 2 倍频率的高频成分}}$$
    
5. **低通滤波 (LPF)：** 滤波器会把后面那个 $2\omega_c$ 的高频垃圾全部滤掉，只剩下最前面的 **$x_c(t)$**。
    

---

### 📝 笔记小结

- **调制：** 基带分量 $\times$ 高频载波 $\rightarrow$ **频谱搬高**。
    
- **解调：** 已调信号 $\times$ 同频载波 $+$ 低通滤波 $\rightarrow$ **频谱搬低（还原）**。
